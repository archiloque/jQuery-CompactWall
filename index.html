jQuery CompactWall

CompactWall is jQuery plugin that will organize a set of blocks to use the minimum vertical space.

It is a variation of the masonry plugin to be used the vertical space is more important than keeping the blocks ordered.

Usage

<div id="container">
    <div class="block">...</div>
    <div class="block">...</div>
    <div class="block">...</div>
    ...
</div>

$(function(){
    // #container is the container
    // .block are the blocks to be optimized inside the container
    $('#container').compactWall($('.block'));
});
Note that the block should have a know size to be reorganized, for example if you use images in your block you have to specify their size in the css.
